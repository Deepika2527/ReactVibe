React is an Open Source JavaScript Library used for building User Interfaces (UI), especially Single Page Applications (SPA). Developed by Jordan Walke at Facebook. Key Features of React 1️⃣ Single Page Application (SPA) A Single Page Application loads only one HTML page, and content updates dynamically without refreshing the entire page. Example websites: Gmail Facebook Netflix Only the required part of the page updates. 2️⃣ Component-Based Architecture React applications are built using components. A component is a small reusable piece of UI. Example Structure: App ├── Header ├── Home ├── Products └── Contact Example files: Home.jsx Products.jsx Contact.jsx Usage: Advantages: Reusable Easy to maintain Organized code 3️⃣ Virtual DOM DOM = Document Object Model. Normally when a change happens: Real DOM 100 elements → change 1 element → entire DOM updates But React uses Virtual DOM. Process: Real DOM = 100 elements Change = 1 element Virtual DOM compares Old vs New Only the changed element updates Result: Faster updates Better performance 4️⃣ One Way Data Binding React follows Unidirectional Data Flow.
